YOLO segmentation annotation generator

How to use:

1. Select a folder with PNG sprites or sprite sheets.
2. Optionally change dataset name, alpha threshold and grid size.
3. Click "GENERATE DATASET" and wait for ZIP download.

The generator splits multi-frame PNGs using OpenTibiaLibrary PNG metadata when available, then auto-detects the crop from PNG height: 32->32x32, 64->64x64, 128->128x128. Manual grid selection is still available for special cases.

Output ZIP contains YOLO-seg labels, cropped training images, classes list and dataset metadata.

FILES ARE NOT SEND TO SERVER! EVERYTHING IS GENERATED ON YOUR COMPUTER.
